Proxmox VE (Virtual Environment) adalah sebuah distro Linux virtualisasi berbasis Debian (64 bit) yang mengusung OpenZV dan KVM, dengan KVM kita tidak hanya bisa menginstall linux saja akan tetapi juga bisa Operating system windows pun bisa kita instal. istimewa dari proxmox adalah kemudahan dalam installasi dan administrasi berbasis Webbase.

Keuntungan menggunakan Proxmox sebagai berikut :
1. Kinerja sangat baik
2. Instalasi yang telah dioptimalkan, sehingga lebih cepat.
3. Mudah dalam manajemen.

Beberapa hal yang perlu diperhatikan :
1. Proxmox tersedia hanya untuk hardware 64-bit.
2. Pada Proses installasi proxmox akan menggunakan seluruh ruang storage yang ada, ini artinya Anda harus menyediakan 1 Server/Komputer dengan penyimpanan khusus hanya untuk proxmox saja.

Yang perulu di persiapkan :
1. Pc/Laptop
2. software proxmox download 3. VirtualBox download 4. Doa yang pasti

Cara Install Proxmox di VirtualBox :
1. Buat Projek baru di VirtualBox
2. Memory kasi 2048 MB atau 2GB 3. Harddisk beri 20GB 4. Setelah di create, jangan lupa untuk ke pengaturan untuk mengisi OS dari proxmoxnya yang akan kita gunakan install setting-storage 5. Jangan lupa juga untuk ke pengaturan networknya, kita menggunakan Bridged Adapter, pergi ke setting - network 6. Setelah itu kita jalankan mesin yang kita buat tadi, maka akan tampil seperti gambar di bawah ini. pilih install proxmox VE 7. Pilih hardisk yang akan di gunakan proxmoxnya. 8. Pilih negara dan time zone, keyboard 9. Lalu Isikan password dan email (gunakan pasword yang kuat kaya cinta mu padanya)
10. Kita atur jaringan, ubah ip address (ip yang belum ada yang pakek), lalu klik next dan install
11. Proses instalasi, ditunggu ya kawan 12. Setelah install dan di reboot maka akan tampil seperti ini, maka yang harus kamu lakukan me-remove disknya, tutup kembali dan jalankan. 13. Lalu salin ip dan port yang diberikan setelah jalankan ulang, dan login ip tersebut maka akan masuk ke panel proxmox
